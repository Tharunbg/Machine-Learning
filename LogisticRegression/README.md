# 🚀 Logistic Regression from Scratch (End-to-End ML Pipeline)

This project is a **complete machine learning pipeline** that implements **Logistic Regression from scratch using Python** — without relying on core implementations from libraries like `scikit-learn`.

It demonstrates a deep understanding of:

- Machine Learning fundamentals  
- Optimization techniques (Gradient Descent)  
- Model evaluation and persistence  
- End-to-end ML workflow design  

---

## 📌 Project Overview

This repository builds a **custom Logistic Regression model** and applies it to the classic **Iris Dataset** for **binary classification**.

### 🔹 The pipeline includes:

- Model implementation (from scratch)  
- Training scripts  
- Evaluation scripts  
- Model saving/loading  
- Decision boundary visualization  

---

## 🧠 Core Features

### ✅ Custom Logistic Regression Implementation
- Sigmoid activation function  
- Mini-Batch Gradient Descent  
- Learning rate optimization  
- Weight & bias updates  

### ✅ Early Stopping Mechanism
- Prevents overfitting  
- Monitors validation loss  
- Stops training when no improvement  

### ✅ Model Persistence
- Save trained weights (`.npz`)  
- Reload model instantly for evaluation  

### ✅ Visualization
- Decision boundary plots using `mlxtend`  
- Clear separation of classes

## 📈 Example Output

- Decision boundary separating classes  
- Accuracy score on test dataset  
- Reduced overfitting due to early stopping
